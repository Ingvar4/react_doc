## Настройка редактора

Ознакомьтесь с нашими [рекомендуемыми редакторами](/docs/setup/editor-setup) и узнайте, как настроить их для работы с React.

## Использование TypeScript

TypeScript — популярный способ добавления определений типов в кодовые базы JavaScript. [Узнайте, как интегрировать TypeScript в ваши проекты React](/docs/setup/typescript).

## Инструменты разработчика React

Инструменты разработчика React — это расширение для браузера, которое может проверять компоненты React, редактировать свойства и состояние, а также выявлять проблемы с производительностью. Узнайте, как его установить [здесь](/docs/setup/react-developer-tools).

## Компилятор React

Компилятор React — это инструмент, который автоматически оптимизирует ваше приложение React. [Узнайте больше](/docs/react-compiler).
