---
title: useImperativeHandle
description: useImperativeHandle — это React-хук, позволяющий настраивать обработчик, отображаемый в виде ссылки.
---

Вызовите метод `useImperativeHandle` на верхнем уровне вашего компонента, чтобы настроить предоставляемый им дескриптор ссылки:

```js
import { useImperativeHandle } from 'react';

function MyInput({ ref }) {
  useImperativeHandle(ref, () => {
    return {
      // ... your methods ...
    };
  }, []);
  // ...
```

## Параметры

- ref: Ссылка, полученная вами в качестве свойства компонентом MyInput.
- createHandle: Функция, которая не принимает аргументов и возвращает дескриптор ссылки, который вы хотите сделать доступным. Этот дескриптор ссылки может иметь любой тип. Обычно вы возвращаете объект с методами, которые хотите сделать доступными.
- Необязательные зависимости: Список всех реактивных значений, на которые ссылается код функции createHandle. Реактивные значения включают props, state, а также все переменные и функции, объявленные непосредственно в теле компонента. Если ваш линтер настроен для React, он проверит, правильно ли указано каждое реактивное значение в качестве зависимости. Список зависимостей должен содержать постоянное количество элементов и быть записан непосредственно в коде, например, [dep1, dep2, dep3]. React сравнит каждую зависимость с ее предыдущим значением, используя сравнение Object.is. Если перерисовка привела к изменению какой-либо зависимости или если вы опустили этот аргумент, ваша функция createHandle будет выполнена повторно, и вновь созданный дескриптор будет присвоен ссылке.
